# Admin Dashboard UI (HTML & CSS)

Admin Dashboard UI adalah project frontend statis yang dibuat sebagai media pembelajaran desain tampilan admin menggunakan HTML dan CSS.
Project ini berfokus pada pemahaman struktur layout, penggunaan CSS untuk styling, serta perancangan antarmuka admin dashboard yang rapi dan mudah digunakan.

Project Purpose
1. Tujuan pengembangan project ini adalah:
2. Mempelajari dasar-dasar HTML dan CSS
3. Memahami struktur layout dashboard admin
4. Menerapkan teknik styling CSS
5. Melatih pembuatan tampilan admin panel secara manual tanpa framework
6. Menjadi dokumentasi awal perjalanan belajar frontend development

Features
1. Fitur tampilan yang tersedia pada project ini antara lain:
2. Layout dashboard admin
3. Sidebar navigasi
4. Header / top navigation
5. Halaman data (tabel statis)
6. Form input statis
7. Tampilan responsif dasar (CSS)

Tech Stack
1. HTML5
2. CSS3
3. Tanpa framework CSS
4. Tanpa JavaScript (fokus tampilan)

Project Background
Project ini dibuat sekitar 2023 sebagai latihan awal dalam memahami:
1. Struktur HTML
2. Pengaturan layout menggunakan CSS
3. Konsep UI untuk admin dashboard
4. Project ini menjadi dasar sebelum melanjutkan ke framework frontend dan backend yang lebih kompleks.

Notes
1. Project ini bersifat statis (frontend only)
2. Tidak memiliki fungsi backend
3. Dibuat untuk tujuan pembelajaran
4. Tidak digunakan untuk production
